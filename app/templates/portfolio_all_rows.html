Stocks, ETFs, and Currencies: Daily Reports on E-mail

I use this application for daily monitoring of stocks and currencies that interest me. The main Python script works as follows:

  1. It gets a list of stocks or ETF tickers from the Google Spreadsheet.
  2. For each position, it downloads the recent Open-High-Low-Close prices data. Currently, the source of data is Alpha Vantage.
  3. Using the obtained data, the system generates a report with charts and sends it to my email.
  4. You can see the list of tickers and their recent charts below.

The user can effortlessly change the set of items in the watchlist. The only needed device is a smartphone that works with Google Spreadsheets.
