Django Crispy Forms Advanced Usage Example

This project aims to demonstrate the creation and processing of the large and complex form in Django using django-crispy-forms. You can edit the names of data schemes, add new schemes, and delete them. Also, edit columns, including changing their types and parameter values.

Don't be surprised when your editing results disappear. The schemes and their columns are reset every 15 minutes using a Celery task.
